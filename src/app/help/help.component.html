Explainable-Admet Help


1.The homepage contains nine pull-down menus: home, prediction, optimization, search, document, help and contact in the toolbar, and users can browse relevant entries by clicking their sub-menus. The corresponding entries will be displayed.

2.In the prediction page, users can input a valid Smiles or draw the compound structure through the JME applet window by selecting the corresponding mode. Then click the “submit” button to deliver a query molecule to Explainable-Admet Server. Users will get the predicted value of 41 properties of this compound and its interpretation diagram.

3.The prediction result is divided into two parts.The first part includes several common attributes that can be directly calculated and an "optimize" button that can jump directly to the optimization page.And the second part includes the calculated predicted value ​​of 41 ADMET properties and the corresponding explanatory diagrams.

4.In the optimize page, users first need to select a property and threshold that want to be optimized, and then input a valid smiles. Click the “submit” button to deliver a query molecule to Explainable-Admet Server. Users will get the Optimization results.

5.The optimization result is also divided into two parts.The first part includes calculated RA score and options for re-optimization and the "predict" button to predict the properties of the compound. If you want to optimize again, just click on the structure on the left to modify it.And the second part includes the structure of the compound optimized by using bioelectronic isosteres and the variable value.If you want to know the predicted value of 41 properties of the changed structure, click on the picture to calculate. And you can download the corresponding file at the bottom of the table.

6.In the search page, users first need to select a search mode(precise or range search),precise search just need you to input vaild Smiles, and then you will get the corresponding data from our database if it exists. For range search, you need to select an attribute and the range of the corresponding attribute value, and then get compounds within the range from the database.

7.We display the specific data of 41 models on the document page for users to use and reference. If you have any questions about the data, You can contact us through the email on the contact page.

8.Explainable-Admet was only test on the latest version of browser, so please try the latest browser (google chrome , firefox, etc) if you have any display problems.

9.If there are any comments or suggestions, please contact Dr. Jianping Lin at jianpinglin@nankai.edu.cn. For more specific information, please see the contact page.
